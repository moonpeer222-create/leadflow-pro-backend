# LeadFlow Pro API

Backend for LeadFlow Pro SaaS.

## Deploy on Render
1. Push to GitHub
2. Connect to Render
3. Deploy with Free plan

## API Endpoints
- POST /api/auth/login
- POST /api/auth/register
- POST /api/scraper/google-maps
- POST /api/scraper/facebook
- POST /api/scraper/linkedin
- GET /api/leads
